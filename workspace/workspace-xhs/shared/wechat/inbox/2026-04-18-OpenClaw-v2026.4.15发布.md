**版本：** v2026.4.15
**发布地址：** https://github.com/openclaw/openclaw/releases/tag/v2026.4.15

### 主要新功能

1. **Anthropic/Models 更新**
   - 默认模型选择优化，Claude Opus 4.7 内置图像理解
   - Claude CLI 默认值更新

2. **Google/TTS：Gemini 语音合成**
   - 支持 Gemini TTS，提供语音选择、WAV 回复输出、PCM 电话输出
   - 含完整设置和文档指引

3. **控制面板新功能**
   - Model Auth 状态卡片：显示 OAuth token 健康度和 API 限流压力
   - Token 即将过期时主动提醒

4. **Memory/LanceDB 云存储支持**
   - 记忆索引支持远程对象存储，不再局限于本地磁盘

5. **GitHub Copilot 记忆搜索集成**
   - 支持 Copilot embedding provider
   - 插件可复用传输层，支持远程覆盖和 token 刷新

6. **本地模型优化**
   - 新增 experimental `localModelLean` 选项，移除浏览器/Cron/Message 等重量级默认工具，降低 prompt 大小

7. **打包优化**
   - 插件运行时依赖本地化，发布包更精简

8. **文档展示页更新**
   - 可扫描的 Hero 区、完整章节跳转链接、社区案例响应式视频网格

### Bug 修复（重点摘要）
- 修复本地媒体工具的信任边界问题
- 修复 Docker pnpm v10+ 虚拟存储路径构建问题
- 修复 Matrix E2EE 密码less bot 引导保守性问题
- 修复 Cron 公告 NO_REPLY 末尾静默哨兵泄漏问题
- 修复 Dreaming 阶段日记写入路径（默认改为 separate 模式）
